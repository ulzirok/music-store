# Music Generator App (Itransition Task 5)

Веб-приложение для генерации детерминированных данных о музыке с поддержкой локализации, бесконечного скролла и процедурного синтеза аудио и пагинация.

**Ссылка на деплой:** [https://enchanting-centaur-aeb498.netlify.app/](https://enchanting-centaur-aeb498.netlify.app/)

## Технологии (Stack)
- **Frontend:** React, React-Bootstrap, Tone.js
- **Backend:** Node.js, Express, Faker.js.

##  Основные возможности
- Детерминированная генерация: Один и тот же Seed всегда генерирует одинаковые данные (названия, артистов, обложки и мелодии). 
- 64-битный Seed: Поддерживаются очень большие значения seed - передаются как строка, без потери точности JavaScript (Это позволяет: точно воспроизводить один и тот же результат и использовать seed как «ID генерации».
- Многоязычность (i18n): Приложение поддерживает локализацию контента и интерфейса для 5 регионов.
- Вероятностная логика лайков: Поддерживаются дробные средние значения лайков - например, 0.5 лайка на песню (Реализовано через алгоритмы распределения вероятностей, а не простое округление).
- Процедурное аудио (Tone.js): Для каждой песни на лету генерируется музыкально связное аудио-превью (создаётся прямо в браузере).
- Два режима отображения: Table View (пагинация, раскрытие деталей песни) и Gallery View (бесконечная прокрутка, подгрузка данных по мере скролла)

## Backend cold start
Бэкенд данного проекта задеплоен на бесплатном тарифе и может находиться в спящем режиме. При первом обращении к серверу после простоя запуск может занять до 30–60 секунд. Если приложение не отвечает сразу — пожалуйста, подождите немного и обновите страницу.
